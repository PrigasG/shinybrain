# Build edge candidates from resolved references

Build edge candidates from resolved references

## Usage

``` r
build_edges(references, nodes, symbols, project_id)
```

## Arguments

- references:

  sb_reference tibble (resolved)

- nodes:

  sb_node_candidate tibble

- symbols:

  sb_symbol tibble

- project_id:

  Project ID

## Value

sb_edge_candidate tibble
